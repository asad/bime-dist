Quick Start

Embed BIME in any web page with 3 steps:

<!-- 1. Include scripts (from GitHub Pages CDN) -->
<script src="https://asad.github.io/bime-dist/editor/Molecule.js"></script>
<script src="https://asad.github.io/bime-dist/editor/Layout.js"></script>
<script src="https://asad.github.io/bime-dist/editor/Templates.js"></script>
<script src="https://asad.github.io/bime-dist/editor/SmilesParser.js"></script>
<script src="https://asad.github.io/bime-dist/editor/SmilesWriter.js"></script>
<script src="https://asad.github.io/bime-dist/editor/MolfileWriter.js"></script>
<script src="https://asad.github.io/bime-dist/editor/Renderer.js"></script>
<script src="https://asad.github.io/bime-dist/editor/History.js"></script>
<script src="https://asad.github.io/bime-dist/editor/Tools.js"></script>
<script src="https://asad.github.io/bime-dist/editor/CipStereo.js"></script>
<script src="https://asad.github.io/bime-dist/editor/SmartsParser.js"></script>
<script src="https://asad.github.io/bime-dist/editor/SmartsMatch.js"></script>
<script src="https://asad.github.io/bime-dist/editor/SmartsWriter.js"></script>
<script src="https://asad.github.io/bime-dist/editor/ImageExport.js"></script>
<script src="https://asad.github.io/bime-dist/editor/MolEditor.js"></script>

<!-- 2. Add a container -->
<div id="editor" style="width:100%;height:500px"></div>

<!-- 3. Create the editor -->
<script>
var editor = new MolEditor('editor', '100%', '500px', {
    options: 'hydrogens'
});
editor.readGenericMolecularInput('c1ccccc1');  // Load benzene
</script>

That's it. No build step, no npm, no bundler. Works from a local file or any web server.

SMILES & Validation

BIME includes a strict SMILES parser that validates input and reports detailed errors:

// Valid molecule
editor.validateSmiles('c1ccccc1');
// { valid: true, atoms: 6, bonds: 6, smiles: 'c1ccccc1', error: null, warnings: [] }

// Unclosed ring
editor.validateSmiles('C1CCC');
// { valid: false, error: 'Unclosed ring 1 opened at position 1' }

// Valence violation (warning, not error)
editor.validateSmiles('C(C)(C)(C)(C)C');
// { valid: true, atoms: 6, warnings: ['Valence violation: atom C has 5 bonds but max standard valence is 4'] }

// Unknown atom
editor.validateSmiles('XYZ');
// { valid: false, error: 'Unknown atom symbol "X" at position 0' }

// Reaction SMILES
editor.validateSmiles('CCO>>CC=O');
// { valid: true, atoms: 6, bonds: 4 }

// Atom-mapped reaction
editor.validateSmiles('[CH3:1][OH:2]>>[CH2:1]=[O:2]');
// { valid: true, atoms: 4 }

SMARTS Substructure Search

Search for substructure patterns using SMARTS notation. BIME includes a VF2-based matching engine.

// Match a SMARTS pattern against the current molecule
var count = editor.matchSmarts('[#6]~[#7]');  // carbon bonded to nitrogen
// Returns number of matches found; matching atoms are highlighted in teal

// Programmatic matching
var mol = new Molecule();
SmilesParser.parse('Cn1c(=O)c2c(ncn2C)n(C)c1=O', mol);  // caffeine

var query = new Molecule();
SmartsParser.parse('[#7]', query);  // any nitrogen

var mappings = SmartsMatch.match(mol, query);  // all nitrogen positions
console.log(mappings.length);  // 4 (caffeine has 4 nitrogens)

// Highlight matches
SmartsMatch.highlightMatches(mol, query);  // sets atom.highlighted = true

Image Export (SVG / PNG)

Export molecules as publication-quality images.

// SVG export (self-contained, no external CSS)
var svg = ImageExport.toSVG(editor.molecule, {
    width: 400, height: 300,
    padding: 40,
    background: '#ffffff',
    bondWidth: 2.5,       // thicker for print
    fontSize: 16,         // larger for print
    showHydrogens: true,
    watermark: true       // "Generated by BIME" in corner
});

// Download as SVG file
ImageExport.downloadSVG(svg, 'caffeine.svg');

// PNG export at 4x resolution (publication quality)
ImageExport.toPNG(editor.molecule, { scale: 4 }).then(function(blob) {
    ImageExport.downloadPNG(blob, 'caffeine.png');
});

// Copy to clipboard (for pasting into documents)
ImageExport.copyToClipboard(editor.molecule);

// Print-ready SVG (thicker lines, larger fonts, CMYK-safe colours)
var printSvg = ImageExport.toPrintSVG(editor.molecule);

// Batch export (1181 molecules → 1181 SVGs)
var svgs = ImageExport.batchSVG(COMMON_MOLECULES, { width: 300, height: 200 });

Molecule Naming

BIME identifies molecules by matching against a built-in offline database of 1181 PubChem-canonical compounds (drugs, amino acids, nucleotides, vitamins, common reagents). No external network calls — your structure never leaves the page.

// Auto-name from 1181 built-in molecules
editor.readGenericMolecularInput('CC(=O)Oc1ccccc1C(=O)O');
console.log(editor.getMolName());  // "Aspirin" (matched from local database)

// Set name manually
editor.setMolName('My Custom Molecule');

// Local-only auto-name (Promise-based; resolves from the built-in database
// or returns null if no match)
editor.autoName().then(function(name) {
    console.log(name);
});

The molecule name is displayed on the drawing canvas. Double-click the name to edit it. The name is included in MOL file exports (V2000 cols 1-80 of the title line, V3000 M V30 NAME) and SVG images.

Note: earlier alpha builds queried cactus.nci.nih.gov for IUPAC names. That third-party channel was removed in v1.0.1; getIUPACName() and related methods are retained as no-op stubs for backwards compatibility but no longer make network calls.

Reactions & Atom Mapping

Draw reactions with the arrow tool and map atoms between reactants and products.

// Load a reaction SMILES
editor.readGenericMolecularInput('CCO>>CC=O');

// Load with atom-atom mapping
editor.readGenericMolecularInput('[CH3:1][OH:2]>>[CH2:1]=[O:2]');

// Retrosynthetic arrow
editor.readGenericMolecularInput('CC=O=>CCO');

// Multi-component reaction (with + signs between fragments)
editor.readGenericMolecularInput('CC(=O)O.OCC>>CC(=O)OCC.O');

Use the Arrow tool to draw reaction arrows, and the Map tool to click atom pairs for mapping.

Stereochemistry (CIP R/S, E/Z)

BIME assigns CIP stereochemistry labels using the Cahn-Ingold-Prelog priority algorithm.

// Load a chiral molecule
editor.readGenericMolecularInput('[C@@H](F)(Cl)Br');

// CIP labels are computed automatically
// Enable display: editor.options('showcip');

// Programmatic access
CipStereo.assign(editor.molecule);
editor.molecule.atoms.forEach(function(atom) {
    if (atom.cipLabel) console.log(atom.symbol + ': ' + atom.cipLabel);
    // "C: R"
});

// E/Z for double bonds
editor.molecule.bonds.forEach(function(bond) {
    if (bond.cipLabel) console.log('Bond: ' + bond.cipLabel);
    // "Bond: E" or "Bond: Z"
});

Testing & Auditing

BIME ships with a 427-test regression suite that runs in plain Node with zero dependencies in roughly 250 ms.

# Full regression suite (parser, writer, SMARTS, VF2, MCS,
# aromaticity, hydrogen counting, round-trip, history,
# torture inputs, AAM, charge/isotope round-trip)
node tools/run-tests.js

# Database integrity audit (Hückel + ring-size + peroxide
# consistency check across all 1181 built-in molecules)
node tools/audit-aromatic.js

The tests are organised one file per concern under tests/:

FileCoverage
test_smiles_parser.jsAtom / bond / ring / branch / stereo parsing, error reporting
test_smiles_writer.jsCanonical Morgan ranking, charge-aware aromaticity
test_smarts_match.jsLogical primitives, recursive SMARTS, ring queries
test_substructure_vf2.jsVF2 subgraph isomorphism with atom / bond constraints
test_mcs.jsMaximum common substructure with timeouts
test_aromaticity.jsHückel 4n+2, charged carbocycles, saturated-C rejection
test_hcount.jsImplicit hydrogen counting and valence rules
test_round_trip.jsParse → write → parse fidelity for canonical and metadata fields
test_history.jsUndo / redo ring buffer; reactionPlusSigns preservation
test_chem_torture.jsZwitterions, polycyclics, charged aromatics, reaction SMILES
test_aam_round_trip.jsAtom-atom mapping survives MOL V2000 / V3000 / RXN
test_charge_iso_round_trip.jsCharge, isotope, and radical preservation through I/O

The aromaticity audit tool catches mass-canonicalisation regressions (aromatic ribose, peroxides, β-lactams, cyclohexane, etc.) and reports 0 issues across all 1181 molecules in the current database.

Architecture

BIME is 15 single-responsibility modules totalling ~10,300 lines:

editor/
  Molecule.js        Molecular graph (atoms, bonds, rings, components)
  Layout.js          2D coordinate generation (SSSR, fused rings, chains)
  Templates.js       Scaffold templates (benzene, indole, steroid...)
  SmilesParser.js    Strict SMILES parser with validation
  SmilesWriter.js    Canonical SMILES writer (Morgan, Huckel)
  MolfileWriter.js   MOL V2000/V3000 export
  Renderer.js        SVG renderer (atoms, bonds, arrows, maps)
  History.js         Undo/redo stack
  Tools.js           10 interactive drawing tools
  CipStereo.js       CIP R/S and E/Z stereochemistry
  SmartsParser.js    SMARTS pattern parser
  SmartsMatch.js     VF2 substructure matching
  SmartsWriter.js    SMARTS pattern writer
  ImageExport.js     Publication SVG, hi-res PNG, clipboard
  MolEditor.js       Main editor class (UI, toolbar, events, API)
